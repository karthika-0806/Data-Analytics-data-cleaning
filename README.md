# Task 1: Data Cleaning and Preprocessing

## 📌 Objective

The objective of this task is to clean and preprocess a raw dataset by identifying and handling common data quality issues such as missing values, duplicate records, inconsistent text formats, incorrect data types, and date formatting. The cleaned dataset is then prepared for further analysis or visualization.

---

## 📂 Dataset

**Dataset Used:** Netflix Movies and TV Shows Dataset

The dataset contains information about movies and TV shows available on Netflix, including details such as title, director, cast, country, release year, rating, duration, and genres.

---

## 🛠️ Tools Used

* Microsoft Excel
* GitHub

---

## 📋 Data Cleaning Steps Performed

* Opened the raw dataset in Microsoft Excel.
* Identified and handled missing values by filling appropriate entries where required.
* Removed duplicate records using Excel's **Remove Duplicates** feature.
* Standardized text values by correcting inconsistent capitalization and removing extra spaces.
* Renamed column headers to maintain a clean and consistent naming convention.
* Formatted date columns into a consistent date format.
* Verified data types to ensure numerical, text, and date fields were correctly formatted.
* Saved the cleaned dataset as a new CSV file.

---

## ✅ Outcome

The dataset was successfully cleaned and prepared for future data analysis. After preprocessing, the dataset became more consistent, accurate, and suitable for analysis and visualization.
